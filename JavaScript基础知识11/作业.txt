判断闰年的公式：(year % 4 == 0 && year % 100 != 0 || year % 400 == 0)

（1）能被4整数并且不能被100整数

（2）能被400整数

2000年是不是闰年？闰年 

function isLeapYear(year){
    return year % 4 == 0 && year % 100 != 0 || year % 400 == 0;
}
